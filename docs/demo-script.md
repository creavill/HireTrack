# Hammy Demo Recording Script

## Setup

- Clean database (delete jobs.db)
- Real Gmail account with job alerts
- Browser window: 1280x800
- Recording tool: OBS or macOS screen recording
- Crop to just the browser window

## Shots (30 seconds total)

### Shot 1: Dashboard Empty State (3s)

- Show the dashboard with "No jobs tracked yet"
- Highlight the warm-industrial design, sidebar

### Shot 2: Scan Emails (5s)

- Click "Scan Emails"
- Show the scanning indicator
- Jobs populate the list

### Shot 3: Browse Job List (5s)

- Scroll through the dense job list
- Show score colors, status tags, company logos
- Hover a few rows

### Shot 4: Job Detail Page (5s)

- Click into a job
- Show the two-column layout
- Scroll through description and requirements

### Shot 5: Enrichment (5s)

- Click "Enrich" on a job
- Show the full description replacing the snippet
- Show the score update

### Shot 6: Cover Letter (4s)

- Click "Generate Cover Letter"
- Show the letter appear
- Copy button

### Shot 7: Follow-Up Timeline (3s)

- Scroll to the email activity timeline
- Show classification badges
- Show action item

## Post-Recording

- Convert to GIF (gifski or ffmpeg)
- Max width: 800px
- Target file size: under 5MB
- Place at: docs/demo.gif
- Reference in README: `![Demo](docs/demo.gif)`

## Recording Tips

- Use a smooth, deliberate mouse movement
- Wait for animations to complete before moving to next shot
- Keep clicks visible (slight pause before clicking)
- If the recording tool supports it, add a subtle click indicator
